Features of Shiba
We provide you the best features in Shiba!
"Fun" Module

The Fun module currently has: 8Ball, Shiba (Random shiba images), Hugging, and more to come.

Polls

Polls adds a poll command to your server making it easy to setup a #polls channel and allow users to create polls with other members. (This feature will be tuned depending on what people want, such as making it only go into a #user-polls channel rather then a channel named #polls, or lock it to certain permissions.)

Suggestions

One of many features currently, suggestions adds a suggestion command to your server making it easy to setup a #suggestions channel and accept suggestions from members!
